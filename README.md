# Twitter Offensive Language Detection

A Machine Learning web application that detects whether a tweet contains offensive language.

## Features
- Text preprocessing
- TF-IDF vectorization
- Logistic Regression classifier
- FastAPI backend
- Interactive web interface

## Tech Stack
- Python
- Scikit-Learn
- FastAPI
- HTML/CSS
- NLP

## Run Locally

```bash
pip install -r requirements.txt
python -m uvicorn app:app --reload
